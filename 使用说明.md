# SmartScribe AI 网文生成工具使用说明

## 项目简介

SmartScribe 是一款轻量化 AI 网文创作工具，支持小说生成、续写、设定创作、大纲生成、内容润色等功能，为网文作者提供全方位的创作支持。

## 系统要求

- Node.js 18.0 或更高版本
- MySQL 8.0 或更高版本
- 现代浏览器（Chrome、Edge、Firefox 最新版）

## 安装步骤

### 1. 克隆项目

```bash
git clone <项目地址>
cd SmartScribe
```

### 2. 配置环境变量

建议在 `backend/.env` 配置（或在系统环境变量中配置）：
- `DB_HOST` / `DB_PORT` / `DB_USER` / `DB_PASSWORD` / `DB_NAME`
- `JWT_SECRET`（必填）
- `ADMIN_BOOTSTRAP_KEY`（建议配置，用于初始化首个 admin）
- `DASHSCOPE_API_KEY` / `GLM_AI_KEY` 等（可选，服务端默认 AI 密钥；也可在前端 AI 设置页填写）

### 3. 安装前端依赖

```bash
cd frontend
npm install
```

### 4. 安装后端依赖

```bash
cd backend
npm install
```

### 5. 配置数据库

1. 确保 MySQL 服务已启动
2. 创建数据库 `smart_scribe`
3. 配置 `backend/.env` 文件中的数据库连接信息

### 6. 启动后端服务

```bash
cd backend
npm run dev
```

### 7. 启动前端服务

```bash
cd frontend
npm run dev
```

### 8. 访问应用

打开浏览器，访问 `http://localhost:3002`

> 后端默认 `http://localhost:3001`，前端通过 `/api` 或环境变量配置访问后端。

### 9. 初始化数据库表（首次运行/升级后）

```bash
cd backend
npm run db:init
```

如果你是从旧版本升级到包含 RBAC、AI 请求日志/用量统计的新版本，请额外执行一次：

```bash
cd backend
npm run db:migrate:rbac-usage
```

## 使用方法

### 1. 注册/登录

- 首次使用需要注册账号
- 注册成功后登录系统

### 2. 新建小说

- 登录后进入创作主页
- 点击「新建小说」按钮
- 输入小说名称和简介
- 点击「创建」按钮进入小说创作页面

### 3. 生成章节

- 在小说创作页面点击「生成章节」按钮
- 输入章节标题（选填）和生成提示词
- 点击「生成」按钮，等待AI生成内容
- 生成的内容会自动显示在编辑器中

### 4. 续写章节

- 在小说创作页面点击「续写章节」按钮
- 输入续写要求（选填）
- 点击「生成」按钮，AI会基于当前章节内容进行续写

### 5. 内容润色

- 在小说创作页面点击「内容润色」按钮
- 输入润色要求（选填）
- 点击「生成」按钮，AI会对当前内容进行润色

### 6. 生成设定

- 在小说创作页面点击相应按钮
- 选择设定类型（人物、世界观、道具）
- 输入设定要求
- 点击「生成」按钮，获取AI生成的设定

### 7. 生成大纲

- 在小说创作页面点击「生成大纲」按钮
- 输入小说类型、核心剧情、篇幅要求等
- 点击「生成」按钮，获取AI生成的小说大纲

### 8. AI设置

- 点击顶部导航栏的「AI设置」
- 选择AI平台（阿里云百炼或智谱AI）
- 选择对应平台下的模型
- 点击「保存配置」按钮

### 9. 意见反馈

- 点击底部导航的「意见反馈」
- 选择反馈类型
- 输入反馈内容
- 点击「提交反馈」按钮

### 10. 管理后台（admin）

- 管理员登录后，右上角用户菜单会出现「管理后台」（路径 `/admin`）。
- 首个管理员需要初始化（只用于创建/提升首个管理员账号）：

```bash
curl -X POST "http://localhost:3001/api/admin/bootstrap" ^
  -H "Content-Type: application/json" ^
  -H "x-admin-bootstrap-key: <你的ADMIN_BOOTSTRAP_KEY>" ^
  -d "{\"username\":\"admin\",\"password\":\"admin123456\"}"
```

## 功能说明

### 核心功能

1. **用户模块**：注册、登录、退出登录
2. **小说管理**：新建小说、小说列表、删除小说
3. **AI创作**：章节生成、章节续写、内容润色、设定生成、大纲生成
4. **编辑器**：Draft.js 富文本编辑、内容保存
5. **AI配置**：切换AI平台和模型
6. **通用交互**：Loading状态、流式响应、打字机效果、意见反馈

### 技术特点

- **流式响应**：采用 SSE（Server-Sent Events）技术，实现实时生成效果
- **打字机效果**：模拟人工打字，提升用户体验
- **响应式设计**：适配PC端、平板端和移动端
- **模块化架构**：前端使用 React + TypeScript，后端使用 Node.js + Express

## 常见问题

### 1. 生成内容速度慢

- 检查网络连接
- 尝试切换AI模型
- 减少生成内容的长度要求

### 2. 生成内容不符合预期

- 提供更详细的提示词
- 尝试调整AI模型参数
- 多次生成，选择最优结果

### 3. 登录失败

- 检查用户名和密码是否正确
- 确保网络连接正常
- 尝试清除浏览器缓存

### 4. 数据库连接失败

- 确保MySQL服务已启动
- 检查数据库连接信息是否正确
- 确保数据库 `smart_scribe` 已创建

## 注意事项

- 本工具使用AI生成内容，仅供参考，实际创作还需人工修改和润色
- 请遵守相关法律法规，不要生成违规内容
- 定期备份小说内容，以防数据丢失
